# 실행 과정 추적 (Execution Transparency)

## 기본 동작: 추적 정보 비표시

토큰 효율을 위해 매 응답에 실행 추적 정보를 **포함하지 않습니다**.

## 조회 방법

실행 추적 정보가 필요하면 `/execution-dashboard` 스킬을 사용하세요.
사용자가 "실행추적", "뭐 참고했어?", "어떤 스킬 썼어?" 등으로 요청하면 해당 스킬이 트리거됩니다.

## 예외: 명시적 요청 시

사용자가 "추적 정보 보여줘", "실행 내역 포함해줘" 등을 요청하면 해당 턴부터 아래 형식으로 표시합니다:

---

### 실행 추적

**Skill 트리거**:
- `/{skill-name}` (자동/수동) — {설명}

**참조 Rule**:
- `.claude/rules/{rule-name}.md` — {용도}

**Hook 실행**:
- `{HookType}`: {script-name} {결과}

**Agent 호출**:
- `{agent-type}` — {작업 내용}

**참조 문서**:
- `{file-path}` — {용도}

---

표시가 활성화된 후 "추적 정보 생략해줘"라고 요청하면 다시 비표시로 전환합니다.

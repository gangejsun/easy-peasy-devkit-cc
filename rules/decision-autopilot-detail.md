# Decision Autopilot — L2/L3 상세 (조건부 로드)

> 본 파일은 자율 레벨 L2/L3 동작과 강등 모니터링 상세를 다룹니다.
> 현재 L0~L1 사용자에게는 매 세션 불필요하므로 paths를 `dev/docs/digital-twin/**`로 한정.
> 본 규칙의 핵심 동작(분기점 인식·듀얼 추천·자가 점검)은 [decision-autopilot.md](.claude/rules/decision-autopilot.md)에서 매 세션 로드됩니다.

## L2 동작: 자율+확인

분기점 발생 시:

1. 의도를 내부 추론 (Step 0)
2. 🧠 AI 독립 추천과 👤 프로파일 추천을 의도 기반으로 내부 산출
3. **일치 시**: 프로파일 기반으로 자율 결정 + 결과 보고
4. **불일치 시**: 프로파일 추천으로 결정하되, 불일치 사실을 보고
5. 사용자 거부 시 즉시 변경 + `autonomy-validation.md`에 불일치 기록

## L3 동작: 자율+예외

분기점 발생 시:

1. 의도를 내부 추론
2. 🎯 **일치 분기점**: 보고 없이 자율 결정 (의도 내부 기록)
3. 👤≠🧠 **불일치 분기점**: 프로파일 추천으로 결정하되 사후 보고
4. ❓ **매핑 불가 분기점**: 프로파일 태그에 매핑 안 되는 새 유형 → 사용자에게 질문

## 승급 조건

| 전환    | 일치율 | 의도 정확도 | 누적 건수 | 연속 세션       |
| ------- | ------ | ----------- | --------- | --------------- |
| L1 → L2 | 85%+   | 90%+        | 20건+     | 5세션 연속 충족 |
| L2 → L3 | 95%+   | 95%+        | 40건+     | 5세션 연속 충족 |

## 강등 모니터링 (L2+ 지속 검증)

### 미달 판정

| 기준            | 값                                                                             |
| --------------- | ------------------------------------------------------------------------------ |
| **미달 트리거** | 세션 내 사용자 명시적 거부 **2건+**                                            |
| **명시적 거부** | "아니", "그거 말고", "다시 해줘" 등 자율 결정 번복. 침묵/무반응은 카운트 안 함 |

### 강등 프로세스

| 조건            | 동작                                                   |
| --------------- | ------------------------------------------------------ |
| 1세션 미달      | 경고 기록. 강등 없음                                    |
| 2세션 연속 미달 | 1단계 강등 (L3→L2 또는 L2→L1) + 프로파일 재보정 트리거 |
| 강등 후 재승급  | 다시 승급 조건 충족 시 재승급 가능, 카운터 리셋        |

> **설계 원칙**: 승급(5세션)보다 강등(2세션)이 민감 — 안전 우선. "침묵=동의" 편향 회피를 위해 명시적 거부 횟수만 카운트.

## 자가-진화 루프 예외 (재확인)

자가-진화 루프의 모든 단계는 L2/L3에서도 항상 L1 동작(사용자 승인 필수) 유지.

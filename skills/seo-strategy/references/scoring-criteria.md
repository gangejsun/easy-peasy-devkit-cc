# SEO Scoring Criteria

## Mode 1: Article SEO Scoring (100점 만점)

### 1. Keyword Optimization (30점)

| 항목                         | 배점 | 기준                                          |
| ---------------------------- | ---- | --------------------------------------------- |
| Title에 타겟 키워드 포함     | 5    | 제목 앞부분 50% 이내 = 5점, 후반부 = 3점      |
| H1에 타겟 키워드 포함        | 3    | 포함 = 3점, 미포함 = 0점                      |
| H2에 타겟/LSI 키워드 포함    | 4    | 50%+ = 4점, 25%+ = 2점                        |
| 첫 100단어에 키워드 포함     | 3    | 포함 = 3점                                    |
| 마지막 100단어에 키워드 포함 | 2    | 포함 = 2점                                    |
| 키워드 밀도 (1-3%)           | 5    | 1-3% = 5점, 0.5-1% or 3-5% = 3점, 그 외 = 1점 |
| LSI 키워드 사용 (5개+)       | 5    | 10+ = 5점, 5-9 = 3점, <5 = 1점                |
| 키워드 변형 사용             | 3    | 동의어/변형 3+ = 3점                          |

### 2. On-Page SEO (25점)

| 항목                         | 배점 | 기준                           |
| ---------------------------- | ---- | ------------------------------ |
| Title Tag 길이 (50-60자)     | 4    | 범위 내 = 4점                  |
| Meta Description (150-160자) | 4    | 범위 내 + CTA 포함 = 4점       |
| URL 구조                     | 3    | slug에 키워드 + 3-5단어 = 3점  |
| 이미지 alt 텍스트            | 3    | 모든 이미지에 설명적 alt = 3점 |
| 내부 링크 (3개+)             | 3    | 3+ 관련 내부 링크 = 3점        |
| 외부 링크 (2개+)             | 3    | 2+ 권위 외부 링크 = 3점        |
| 헤딩 구조                    | 3    | 올바른 계층 = 3점              |
| Schema Markup                | 2    | Article/FAQ JSON-LD = 2점      |

### 3. Content Quality (25점)

| 항목               | 배점 | 기준                                      |
| ------------------ | ---- | ----------------------------------------- |
| 단어 수            | 5    | 1500+ = 5점, 1000-1499 = 3점, <1000 = 1점 |
| 가독성             | 4    | Grade 6-8 = 4점, 9-12 = 2점, 13+ = 1점    |
| 단락 길이          | 3    | 평균 3-4문장 = 3점                        |
| 리스트/테이블 사용 | 3    | 2+ 사용 = 3점                             |
| 멀티미디어         | 3    | 3+ = 3점, 1-2 = 2점                       |
| 오리지널 콘텐츠    | 4    | 고유 관점/데이터 = 4점                    |
| E-E-A-T 시그널     | 3    | 저자 정보, 출처 인용, 전문성 증거 = 3점   |

### 4. Technical (20점)

| 항목                 | 배점 | 기준                                       |
| -------------------- | ---- | ------------------------------------------ |
| 모바일 최적화        | 5    | 반응형 + 적절한 폰트/터치 = 5점            |
| 페이지 속도          | 5    | 이미지 최적화 + 렌더링 블로커 최소화 = 5점 |
| HTTPS                | 2    | SSL = 2점                                  |
| Canonical URL        | 3    | 올바른 canonical = 3점                     |
| OG/Twitter 메타      | 3    | 완전한 소셜 메타 = 3점                     |
| 구조화 데이터 유효성 | 2    | 유효한 JSON-LD = 2점                       |

Technical SEO (25점), On-Page SEO (25점), Content (20점), Performance (15점), Mobile (10점), Accessibility (5점).

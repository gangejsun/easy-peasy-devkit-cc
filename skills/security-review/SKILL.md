## 심각도 정의

| 심각도       | 기준                                | 수정 기한      |
| ------------ | ----------------------------------- | -------------- |
| **Critical** | 데이터 유출, 인증 우회, 시크릿 노출 | 즉시           |
| **High**     | 권한 상승, XSS, 입력 검증 부재      | 24시간 내      |
| **Medium**   | 정보 노출, 보안 헤더 미설정         | 다음 릴리스 전 |
| **Low**      | 모범 사례 미준수                    | 백로그         |

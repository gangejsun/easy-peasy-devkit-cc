# 사업 기획 프레임워크

SKILL.md에서 참조하는 사업 기획 관련 프레임워크 및 템플릿입니다.

---

## 1. 5C Analysis 가이드

5C 분석은 사업 환경을 체계적으로 이해하기 위한 프레임워크입니다.

| 요소 | 핵심 질문 | 분석 포인트 |
|------|----------|------------|
| **Company** (자사) | 우리의 핵심 역량은? | 기술력, 인적자원, 자산, 브랜드, 기존 사용자 기반 |
| **Customer** (고객) | 누구의 문제를 해결하는가? | 세그먼트, 니즈, 구매력, 의사결정 방식 |
| **Competitor** (경쟁) | 대안은 무엇이 있는가? | 직접 경쟁, 간접 경쟁, DIY 대안, 전환 비용 |
| **Collaborator** (협력) | 누구와 함께해야 하는가? | 공급자, 유통, 기술 파트너, 플랫폼 |
| **Climate** (환경) | 외부 환경은 어떤가? | PEST(정치/경제/사회/기술), 규제, 트렌드 |

## 2. Lean Canvas 템플릿

| 항목 | 설명 |
|------|------|
| 문제 (Problem) | 타겟 고객이 겪는 상위 3개 문제 |
| 고객 세그먼트 | 타겟 고객 정의 |
| 고유 가치 제안 (UVP) | 한 문장으로 표현하는 차별화 포인트 |
| 솔루션 | 각 문제에 대한 해결 방식 |
| 채널 | 고객에게 도달하는 경로 |
| 수익원 | 수익 모델 |
| 비용 구조 | 주요 비용 항목 |
| 핵심 지표 | 성공 측정 KPI |
| 경쟁 우위 | 쉽게 복제할 수 없는 강점 |

## 3. 가격 책정 프레임워크

| 전략 | 적합 상황 | 예시 |
|------|----------|------|
| 원가 기반 | 원가 산출이 명확 | 제조업 |
| 가치 기반 | 고객 인지 가치가 원가보다 높음 | SaaS |
| 경쟁 기반 | 경쟁 시장, 차별화 어려움 | 이커머스 |
| 프리미엄 | 차별화된 가치, 타겟이 명확 | B2B SaaS |
| 침투 가격 | 시장 점유율 확보 우선 | 신규 진입 |
